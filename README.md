# Introduction to Algorithms

Welcome to the **Introduction to Algorithms** repository! 📘

This repository contains practical implementations and problem‑solving code for core algorithm concepts using C++.  
It’s designed to help learners understand essential algorithm techniques and apply them in real problems.

---

## 📚 Topics Covered

The following algorithmic topics and techniques are included:

- 🔹 Graph Traversal Algorithms(BFS,DFS)
- 🔹 Cycle Detection & Dijkstra
- 🔹 Bellman Ford, Floyd Warshall, DSU
- 🔹 Dynamic Programming   
- 🔹 0-1 Knapsack etc.

Each implementation emphasizes clarity and step‑by‑step logic to support learning and understanding.
